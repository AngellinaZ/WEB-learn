### 执行上下文栈 (Execution context stack，ECS)
 * javascript代码执行顺序： 顺序执行
```javascript 
//变量提升
var foo = function () {

    console.log('foo1');

}

foo();  // foo1

var foo = function () {

    console.log('foo2');

}

foo(); // foo2
```
```javascript 
//函数提升
function foo() {

    console.log('foo1');

}

foo();  // foo2

function foo() {

    console.log('foo2');

}

foo(); // foo2
```
 * 可执行代码 ： 全局代码， 函数代码， eval代码
 * 当执行一个函数的时候，就会创建一个`执行上下文`，并且压入`执行上下文栈`，当函数执行完毕的时候，就会将函数的执行上下文从栈中弹出。
